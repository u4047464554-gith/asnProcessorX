# ASN1C Vendor Source

This directory contains the vendored source code for `asn1c` (The ASN.1 Compiler).

## Provenance

- **Repository**: https://github.com/vlm/asn1c
- **Commit**: `9925dbbda86b436896108439ea3e0a31280a6065`
- **Date**: 2025-11-26 (Vendored)

## Directory Structure

- `repo/`: The Git clone of the upstream repository.
- `patches/`: Local patch files to be applied to the source.
- `bin/`: Compiled binary output (generated by build script).

## Building

See `scripts/build_asn1c.py` for build instructions.
